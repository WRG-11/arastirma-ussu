# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

## Scope

The Contributor Covenant applies to all project spaces, including the
GitHub repository, issue tracker, pull request reviews, and any
official communication channels.

## Reporting

Concerns about contributor conduct should be reported privately to the
maintainer at **yakuphan.yucel11@gmail.com**. All reports are reviewed
and investigated promptly and fairly.

The maintainer follows the
[Community Impact Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
documented in the linked covenant for enforcement decisions.
